- INSTALLATION
1. Ensure **Python 3.+** is installed.  
2. Clone the repository:
--bash--
git clone https://github.com/EngrJaen/Project_Scaper.git
cd business-scraper
pip install -r requirement.txt


- RUN SCRIPT
1. Open a terminal or command prompt in the project folder.
2. Run the scraper:
python scraper.py
3. If a reCAPTCHA appears in the browser:
* Solve it manually.
* Press ENTER in the terminal after solving, as instructed in the terminal output.
4. output and logs
<path_where_you_ran_git_clone>\<repo_folder>\output.json
<path_where_you_ran_git_clone>\<repo_folder>\scraper.log

- LIBRARY USED
1. selenium - Automates browser interactions and scrapes dynamic content generated by JavaScript.
3. json - Saves scraped data to output.json.

- ASSUMPTIONS & LIMITATIONS
1. Each business link results in one record per entity.
2. Requires a working Edge or Chrome browser installed.
3. CAPTCHA cannot be bypassed automatically; user must solve it manually.
4. Network errors or missing elements may occur; basic error handling is included.
5. The website HTML structure is assumed to be consistent; changes may break scraping.
